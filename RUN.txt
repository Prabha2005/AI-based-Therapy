✅ HOW TO RUN THE BACKEND (FastAPI)
📍 Location

Go to the folder where main.py is present
(example: full_api_backend)

🔹 Step 1: Open Command Prompt

Press Windows + R

Type cmd

Press Enter

🔹 Step 2: Go to backend folder
cd C:\Users\poovi\Downloads\AI-based-Therapy-App-main\AI-based-Therapy-App-main\full_api_backend


(Adjust path if needed)

🔹 Step 3: Activate virtual environment (if you have one)

If you created venv:

venv\Scripts\activate


If not, you can skip this.

🔹 Step 4: Run backend server
uvicorn main:app --reload

✅ Backend is running correctly if you see:
INFO:     Uvicorn running on http://127.0.0.1:8000
Application startup complete.


👉 Do NOT close this CMD window

🔹 Optional backend test

Open browser:

http://127.0.0.1:8000/docs


If Swagger opens → backend is ✅

✅ HOW TO RUN THE FRONTEND (Next.js / React)
📍 Location

Go to the folder where package.json exists
(example: AI-based-Therapy-App-main or frontend)

🔹 Step 1: Open a NEW Command Prompt

⚠️ Backend CMD must stay open
Open another CMD window.

🔹 Step 2: Go to frontend folder
cd C:\Users\poovi\Downloads\AI-based-Therapy-App-main\AI-based-Therapy-App-main


(or wherever package.json is)

🔹 Step 3: Install dependencies (only first time)
npm install

🔹 Step 4: Run frontend server
npm run dev

✅ Frontend is running if you see:
Local: http://localhost:3000

🔹 Open frontend in browser
http://localhost:3000


For chatbot page:

http://localhost:3000/chatbot

✅ HOW BOTH WORK TOGETHER (IMPORTANT)
Part	URL
Backend API	http://127.0.0.1:8000
AI endpoint	http://127.0.0.1:8000/ai/chat
Frontend UI	http://localhost:3000
Chatbot page	http://localhost:3000/chatbot

Frontend calls backend internally using fetch.
